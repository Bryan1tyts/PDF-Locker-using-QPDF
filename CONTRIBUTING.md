# Contributing to PDF Locker

Thanks for your interest in contributing! 🎉

## How to Contribute

### Reporting Bugs

If you find a bug, please open an issue with:
- Your OS (Linux/Windows) and version
- Python version (`python3 --version`)
- qpdf version (`qpdf --version`)
- Steps to reproduce the bug
- Expected vs actual behavior

### Suggesting Features

Feature requests are welcome! Please:
- Check existing issues first
- Describe the feature clearly
- Explain why it would be useful

### Pull Requests

1. Fork the repo
2. Create a branch (`git checkout -b feature/amazing-feature`)
3. Make your changes
4. Test on both Linux and Windows (if possible)
5. Commit with clear messages
6. Push and create a PR

### Code Style

- Follow PEP 8 for Python code
- Add comments for complex logic
- Keep functions small and focused
- Test your changes!

### Testing Checklist

Before submitting:
- [ ] Code works on Linux
- [ ] Code works on Windows
- [ ] qpdf command executes correctly
- [ ] GUI displays properly
- [ ] Error handling works
- [ ] No new dependencies (if avoidable)

## Questions?

Feel free to open an issue for any questions!
